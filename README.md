# Student Productivity Tracker (Java)

This is a Java-based console application designed to help students track their study productivity.

## Features
- Add daily study tasks
- Record study sessions
- Track total study hours
- Simple menu-driven interface

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## Purpose
This project was built to solve a real-world problem where students struggle to track their study time and maintain consistency.
